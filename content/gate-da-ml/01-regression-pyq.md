## Q5034

[NAT]
[PYQ: GATE DA 2025]

Consider the dataset $\{(x_1, y_1), (x_2, y_2), \ldots, (x_n, y_n)\}$. We fit the model $y = wx$ (no intercept) using the least squares method, minimizing $\sum_{i=1}^{n}(y_i - wx_i)^2$. Given $\sum_{i=1}^{n} x_i^2 = 14$ and $\sum_{i=1}^{n} x_i y_i = 4$, find the optimal value of $w$ (rounded to 2 decimal places).

**Answer:** 0.29

## Q5035

[NAT]
[PYQ: GATE DA 2026]

Consider the model $y = wx$ (no intercept) trained using stochastic gradient descent with the loss function $L(w) = \frac{1}{2}(y - wx)^2$. Given current weight $w = 10$, a single training point $(x, y) = (10, 9)$, and learning rate $\eta = 0.10$, compute the updated weight $w_{\text{new}}$ after one SGD step (rounded to 1 decimal place).

**Answer:** 9.0
